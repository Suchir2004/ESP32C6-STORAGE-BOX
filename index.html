Flash Firmware

Installation Steps

  • Connect ESP32-C6 via USB.
  • Use Chrome or Microsoft Edge.
  • Click Install.
  • Select the serial port.
  • Wait for flashing to finish.
  • Enjoy your device.

Troubleshooting

  • Use a data USB cable.
  • Install USB drivers if required.
  • Hold BOOT button if connection fails.
  • Try another USB port.
  • Close Serial Monitor before flashing.
